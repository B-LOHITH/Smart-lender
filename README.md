# Smart Lender

Welcome to the Smart Lender project repository. This repository is structured into different phases of the software development lifecycle (SDLC).

## Repository Structure

1. **1. Brainstorming & Ideation**: Initial ideas, brainstorming sessions, and concept notes.
2. **2. Requirement Analysis**: SRS, functional/non-functional requirement specifications.
3. **3. Project Design Phase**: System architecture, UML diagrams, database schemas, and UI mockups.
4. **4. Project Planning Phase**: Gantt charts, schedules, and resource allocation plans.
5. **5. Project Development Phase**: Source code, components, and active development resources.
6. **6.Project Testing**: Test cases, test suites, bug reports, and testing results.
7. **7.Project Documentation**: User manuals, API documentation, and deployment guides.
8. **8.Project Demonstration**: Demo presentations, slides, and recordings.

## Getting Started

Please navigate to the respective directories depending on the current phase of development.
